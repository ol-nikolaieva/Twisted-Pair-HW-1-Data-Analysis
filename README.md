# Статистичний аналіз датасету 

## Опис

Репозиторій містить код і приклади для виконання базового статистичного аналізу датасету за допомогою **pandas**. Завдання:

1. Завантажити обраний датасет за допомогою `pandas`.
2. Підрахувати основні статистичні характеристики для кожної з чисельних ознак:

   * мода, медіана, математичне очікування (середнє);
   * дисперсія;
   * 25%, 50% (медіана) та 75% квантилі.
3. Задача з зірочкою: зробити все вищеперераховане для категорійних змінних.
