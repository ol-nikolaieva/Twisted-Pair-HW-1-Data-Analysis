# Статистичний аналіз датасету (pandas)
Опис
Репозиторій містить код і приклади для виконання базового статистичного аналізу датасету за допомогою pandas. Завдання:
Завантажити обраний датасет за допомогою pandas.
Підрахувати основні статистичні характеристики для кожної з чисельних ознак:
мода, медіана, математичне очікування (середнє);
дисперсія;
25%, 50% (медіана) та 75% квантилі.
Задача з зірочкою: зробити все вищеперераховане для категорійних змінних.
